WebSocket allows you to create real-time applications — chats, games, notifications. Unlike regular HTTP requests, the connection is maintained continuously, and the server can send data to the client on its own. In this article, we will build a chat with rooms using Node.js and the socket.io library.

Installation

Create a project folder and install dependencies 

mkdir websocket-chat
cd websocket-chat
npm init -y
npm install express socket.io

Node.js Server

Create a file server.js and connect Express and socket.io 

const express = require('express');
const http = require('http');
const { Server } = require('socket.io');

const app = express();
const server = http.createServer(app);
const io = new Server(server);

// serve static files from public folder
app.use(express.static('public'));

io.on('connection', (socket) => {
  console.log('New user connected:', socket.id);

  // user joins a room
  socket.on('joinRoom', (room, username) => {
    socket.join(room);
    socket.username = username;
    socket.room = room;
    console.log(`${username} joined room ${room}`);

    // message only for this room
    io.to(room).emit('message', {
      user: 'System',
      text: `${username} has joined room ${room}`
    });
  });

  // handle incoming messages
  socket.on('chatMessage', (msg) => {
    io.to(socket.room).emit('message', {
      user: socket.username,
      text: msg
    });
  });

  // disconnect
  socket.on('disconnect', () => {
    if(socket.username && socket.room){
      io.to(socket.room).emit('message', {
        user: 'System',
        text: `${socket.username} left the room`
      });
    }
  });
});

server.listen(3000, () => {
  console.log('Server running at http://localhost:3000');
});

Client Script

Create public/chat.js and write the client-side logic 

$(function(){
  const socket = io();

  $('#join').click(function(){
    const username = $('#username').val();
    const room = $('#room').val();
    if(username && room){
      socket.emit('joinRoom', room, username);
      $('#login').hide();
      $('#chat').show();
    }
  });

  $('#send').click(function(){
    const msg = $('#msg').val();
    if(msg){
      socket.emit('chatMessage', msg);
      $('#msg').val('');
    }
  });

  socket.on('message', function(data){
    $('#messages').append('<div><b>' + data.user + ':</b> ' + data.text + '</div>');
    $('#messages').scrollTop($('#messages')[0].scrollHeight);
  });
});

socket.io Features

Main events and capabilities
  • connection – triggered when a client connects
  • disconnect – when disconnected
  • socket.join(room) – join a room
  • io.to(room).emit() – send a message to everyone in the room
  • socket.emit() – message to a specific user
  • socket.broadcast.emit() – message to everyone except the sender

Conclusion

We built a Node.js chat with rooms, name-based login, and real-time messages. Socket.io simplifies working with WebSocket and allows building complex systems — chats, notifications, games. You can add authentication, store chat history in a database, and much more.
